Files and Functions Required:

1. shell.c
   - main - The entry point for the shell program.

2. prompt.c
   - print_prompt - Function to display the shell prompt to the user.

3. input.c
   - shell_input - Function to get the user's input from the command line.
   - read_input - Wrapper function to read characters from the command line.
   - _getline - Custom getline function to read user input without using the standard library getline.

4. parse.c
   - parse - Function to parse the command line input into tokens.
   - tokenize - Function to split the input string into an array of strings (tokens).

5. execute.c
   - shell_execute - Function to execute the parsed command.
   - run_command - Function to run a single command with its arguments.
   - builtin - Function to check if a command is a built-in and execute it.
   - execute - Wrapper function to execute a program using execve.

6. path.c
   - find_path - Function to find the full path of a command.
   - handle_PATH - Function to handle command execution with PATH variable.

7. builtins.c
   - builtin_env - Built-in function to print the current environment.
   - builtin_exit - Built-in function to exit the shell.
   - builtin_cd - Built-in function to change the current directory.
   - builtin_setenv - Built-in function to set a new environment variable or modify an existing one.
   - builtin_unsetenv - Built-in function to remove an environment variable.

8. utility.c
   - _putchar - Function to write a single character to stdout.
   - _puts - Function to write a string to stdout.
   - _strcmp - Function to compare two strings.
   - _strdup - Function to duplicate a string.
   - _strlen - Function to calculate the length of a string.

9. error_handling.c
   - handle_errors - Function to handle errors and print appropriate error messages.

10. alias.c
    - handle_alias - Function to implement the alias built-in command.
    - set_alias - Function to create or modify an alias.
    - print_alias - Function to print the list of aliases or a single alias.

11. environment.c
    - update_pwd_env - Function to update the PWD environment variable when changing directories.

12. logical_operators.c
    - handle_logical_operators - Function to handle the && and || shell logical operators.

13. separators.c
    - handle_separators - Function to handle the commands separator ';'.
